论文: Adaptive Differential Denoising for Respiratory Sounds Classification (Interspeech 2025)

核心方法: ADD4RSC 包含三个创新：

Adaptive Frequency Filter - 自适应频率滤波器，学习频谱掩码
Differential Denoise Layer - 差分去噪层，使用差分注意力减少噪声
Bias Denoising Loss - 偏置去噪损失，联合优化分类和鲁棒性
最佳结果: AST 模型达到 65.53% Score (Sp: 85.13%, Se: 45.94%)




复现 AST 结果 (65.53%):

cd ADD-RSC-homework20260604
CUDA_VISIBLE_DEVICES=6 python main.py \
    --model ast \
    --pretrained \
    --data_folder ./data/ICBHI/ICBHI_final_database \
    --audioset_ckpt ./ast-finetuned-audioset-10-10-0.4593.pth \
    --tag ast_add \
    --epochs 50 \
    --batch_size 8 \
    --learning_rate 5e-5 \
    --denoise_d_model 256 \
    --denoise_num_heads 8 \
    --denoise_depth 6 \
    --loss_beta 0.5


复现 ResNet50 结果 (58.97%):

CUDA_VISIBLE_DEVICES=6 python main.py \
    --model resnet50 \
    --tag resnet_add \
    --epochs 50 \
    --batch_size 8


关键参数
参数	说明	默认值
--denoise_d_model	去噪 transformer 隐藏层大小	256
--denoise_num_heads	注意力头数	8
--denoise_depth	去噪 transformer 层数	6
--loss_beta	去噪损失权重	0.5
