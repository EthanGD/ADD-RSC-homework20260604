### AST vs ResNet50 区别

| | ResNet50 | AST (Audio Spectrogram Transformer) |
|---|---|---|
| **架构** | CNN（卷积神经网络） | Transformer（自注意力机制） |
| **预训练** | 随机初始化，从头训练 | 加载 AudioSet 预训练权重，微调 |
| **输入处理** | Mel 频谱图 → 卷积提取局部特征 | Mel 频谱图 → 切 patch → 全局注意力建模 |
| **参数量** | 较小 | 较大（需加载 340MB 预训练权重） |
| **效果** | Score 58.97% | **Score 65.53%**（论文最优） |
| **训练速度** | 较快 | 较慢 |

简单来说：
- **ResNet50** — 轻量快速，但效果稍差，适合快速验证流程
- **AST** — 论文主方法，利用大规模音频预训练，效果最好，是论文报告的 SOTA 结果

如果想**复现论文最优结果**，用 `--model=ast`；如果想**快速跑通流程**，用默认 ResNet50 即可。
