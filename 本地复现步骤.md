
## 本地复现步骤

### 1. 安装依赖
安装 PyTorch 及相关包，推荐版本：`torch==2.0.1, torchaudio==2.0.1, cuda==11.7`

### 2. 下载 ICBHI 2017 数据集
从 [官方挑战页面](https://bhichallenge.med.auth.gr/ICBHI_2017_Challenge) 下载，解压后放到项目目录下，确保路径为：
```
./data/ICBHI/ICBHI_final_database
```

### 3. 下载 AST 预训练权重
从 [Hugging Face](https://huggingface.co/MIT/ast-finetuned-audioset-10-10-0.4593) 下载 `ast-finetuned-audioset-10-10-0.4593.pth`，放到项目目录下

### 4. 启动训练

**使用 ResNet50 作为 backbone（默认）：**
```powershell
python main.py --data_folder='./data/ICBHI/ICBHI_final_database'
```

**使用 AST 作为 backbone：**
```powershell
python main.py --model=ast --audioset_ckpt='./ast-finetuned-audioset-10-10-0.4593.pth' --data_folder='./data/ICBHI/ICBHI_final_database'
```

### 5. 查看结果
训练过程中每个 epoch 会打印 `S_p`、`S_e`、`Score` 指标，最佳模型保存在 `./save/icbhi_<model>_<tag>/best.pth`
